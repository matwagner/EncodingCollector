SelectOrInsertAlias (   $datasource,
  $aliasname,
  $charset,

Tries to find a new alias in a character set. If not found than it will be created. Creates a simplified version of the alias, too if not done already.

This also adds the simplified version of the alias. Relations will be added, too.

Parameters
$datasourceName of the data source
$aliasnameOriginal name of the new alias
$charsetThe character set to which the new alias will be added
$reflistList of references

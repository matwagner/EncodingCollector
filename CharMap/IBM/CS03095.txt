* 1. This character set contains the set of 171 characters populating CPGID 01445 which is the code page for the IBM Advanced Function Printing (AFP) Private Use Area No. 1, defined for use in row FF of the Private User Area Plane 15 (x0F) of ISO/IEC 10646 and Unicode.  Unlike other character sets, there is no assumed SPACE (SP010000) character with this set.
* 
* 2. Nov 16, 2011 - Replaced AFP (IXA0*) GCGIDs with 'Far East' GCGIDs (U0FF*).
* 
* 3. May 7, 2012 - Added four (4) new characters: SF5700H0 (Solid Fill Character, Bottom Half - Halfwidth), SF6100H0 (Solid Fill Character - Halfwidth), SM7800H0 (Left-Right Arrow  Halfwidth) and SM9100H0 (Two Musical Notes - Halfwidth).
